Place ZX Spectrum ROM files here:

  48.rom       - ZX Spectrum 48K ROM (16KB) - Required
  128.rom      - ZX Spectrum 128K ROM (32KB) - Optional
  plus2.rom    - ZX Spectrum +2 ROM (32KB) - Optional
  plus2a.rom   - ZX Spectrum +2A ROM (64KB) - Optional
  pentagon.rom - Pentagon 128K ROM (32KB) - Optional
  trdos.rom    - TR-DOS 5.03/5.04t (16KB) - For TRD/SCL disk images

Amstrad has granted permission for ROM distribution with emulators.
ROMs can be found at: https://worldofspectrum.org/
